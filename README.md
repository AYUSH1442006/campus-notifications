# Campus Notifications System

A React + Material UI campus notification app built for the SRM campus hiring evaluation.

## Tech Stack
- React + Vite
- Material UI
- React Router
- Custom Logging Middleware

## Setup
npm install
npm run dev

App runs on http://localhost:3000

## Features
- All Notifications page with filter and pagination
- Priority Notifications page (Placement > Result > Event)
- Seen/Unseen tracking with localStorage
- Custom logging middleware
- Responsive design (desktop + mobile)
